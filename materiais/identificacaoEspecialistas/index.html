Sistemas computacionais para a identificação de especialistas em ambiente corporativo


Um dos maiores desafios em organizações é administrar o conhecimento dos seus colaboradores, por exemplo, identificando especialistas para uma determinada necessidade de informação ou tarefa. Atualmente, uma das ferramentas utilizadas para tentar resolver este desafio é o conceito de página amarela. Páginas amarelas são implementadas criando e atualizando o perfil de cada pessoa da organização manualmente - utilizando um sistema com banco de dados ou apenas fichas em word. Este tipo de abordagem tem um custo alto e deve ser aprimorada. Este artigo apresenta algumas formas para automatizar este processo e, consequentemente, aumentar a sua eficiência.
