IPM - Sistema para Gestão de Projetos


Este projeto foi executado entre Janeiro de 2001 e Dezembro de 2002. O cliente foi a Ericsson do Brasil. A equipe que desenvolveu este projeto era composta por professores, alunos de pós-graduação e alunos de graduação do Laboratório de Engenharia de Conhecimento da Escola Politécnica da Universidade de São Paulo.

Objetivos

A gestão de projetos complexos exige o suporte de um sistema computacional para que se possa atender aos requisitos de tempo de resposta para análise de dados dos projetos, tomada de decisão, comunicação das decisões e implantação de ações. Esta necessidade justifica a implantação de um sistema informatizado que dê suporte aos processos de Gestão de Projetos de maneira integrada na organização. Os principais objetivos do projeto são: (i) pesquisar e desenvolver um sistema de gestão de projetos que se adeque às necessidades de uma empresa com vários projetos simultâneos de grande porte, (ii) desenvolvimento de um processo formal de gestão de projetos que seja suportado pelo sistema e (iii) desenvolver a competência necessária para a criação e o uso do sistema em ambiente real.
