Tratamento de informações na Web: captação e manutenção de dados, mecanismos de busca, mineração de dados e sistemas de recomendação.


Nesta página são encontrados algumas publicações de alguns trabalhos que envolvem os temas:

  3. Uso de Wikis para Gestão da Informação;
  4. Processamento de Linguagem Natural
  5. Adaptabilidade de sistemas e Sistemas de Recomendação, e;
  6. Web Data Mining.
