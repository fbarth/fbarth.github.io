Recuperação e mineração de informações para domínios de investigação


Este projeto iniciou em Março de 2006, parcialmente financiado pelo CNPq através de duas bolsas RHAE (processo número 520092/06-6) e executado pela Fundação Aplicações de Tecnologias Críticas - Atech. Este projeto foi mantido e executado integralmente pela Fundação Aplicações de Tecnologias Críticas - Atech até outubro de 2009.

Objetivo

O objetivo deste projeto foi desenvolver um sistema de recuperação e mineração de informação projetado para auxiliar a atividade de investigação. Este sistema deve ser capaz de processar diversas fontes de informação e identificar relações entre os itens (i.e., documentos, registros de banco de dados) recuperados. As evidências de relacionamento entre itens devem ser apresentadas por diversos meios, incluindo representações gráficas. A apresentação gráfica dos resultados deve permitir ao investigador construir panoramas visuais de investigação em andamento e, de forma interativa, explorar os dados e monitorar mudanças nos cenários da investigação.

As palavras-chave deste trabalho são:

  1. Recuperação e mineração de informações;
  2. Expansão de Consultas utilizando ontologias;
  3. Reconhecimento de entidades nomeadas;
  4. Algoritmos de agrupamento, e;
  5. Técnicas de visualização de padrões em redes sociais.
